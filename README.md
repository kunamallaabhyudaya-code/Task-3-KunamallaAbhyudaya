# ATM Interface

## Description
The ATM Interface is a Java-based console application that simulates the basic functionalities of an Automated Teller Machine (ATM). The project is developed using Object-Oriented Programming (OOP) concepts and provides a simple, menu-driven interface for users to perform banking operations.

## Features
- Check Account Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Menu-Driven Interface
- Secure Transaction Handling

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class

## Project Structure
```
ATMInterface.java
```

## How to Run

1. Compile the program:
   ```bash
   javac ATMInterface.java
   ```

2. Run the program:
   ```bash
   java ATMInterface
   ```

## Sample Operations
- View Balance
- Deposit Funds
- Withdraw Funds
- Exit Application
